# sprint7-final-project

## Objetivo
Analizar el comportamiento de los clientes de ConectaTel mediante técnicas de limpieza, exploración y segmentación de datos para identificar patrones de uso y generar recomendaciones de negocio.

## Dataset utilizado
El análisis se realizó utilizando la base de datos de usuarios de ConectaTel, que incluye información demográfica y métricas de uso como:
- Edad
- Ciudad
- Tipo de plan
- Cantidad de mensajes
- Cantidad de llamadas
- Minutos de llamada
- Fecha de registro

## Etapas del análisis
1. Exploración inicial de los datos.
2. Identificación y tratamiento de valores nulos.
3. Eliminación de registros duplicados.
4. Análisis descriptivo de variables numéricas.
5. Visualización mediante histogramas.
6. Detección de outliers usando Boxplots e IQR.
7. Segmentación de clientes por nivel de uso.
8. Segmentación de clientes por edad.
9. Generación de insights ejecutivos para stakeholders.

## Principales hallazgos
- La mayoría de los usuarios pertenece al segmento de Uso Medio.
- Los adultos representan el grupo de edad más numeroso.
- Se identificaron outliers en mensajes, llamadas y minutos, pero corresponden a usuarios reales con alto consumo.
- Existen oportunidades para crear planes especializados para usuarios de alto uso.

## Cómo ejecutar el proyecto
1. Descargar el archivo .ipynb.
2. Abrirlo en Google Colab o Jupyter Notebook.
3. Ejecutar las celdas en orden.
